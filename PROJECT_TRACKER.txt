================================================================================
                              PROJECT OVERVIEW
================================================================================

CricSocial is a comprehensive cricket-focused social media platform that combines:
- Social networking features (Instagram-like experience)
- Real-time cricket match tracking and live scoring
- Cricket statistics and analytics
- Tournament management system
- AI-powered cricket features
- Cricket coaching and development tools
- Venue management and discovery

Technology Stack:
- Frontend: React + TypeScript + Vite + Tailwind CSS + Shadcn UI
- Backend: Node.js + Express.js + TypeScript
- Database: In-memory (migrating to PostgreSQL)
- Real-time: Socket.io
- External APIs: RapidAPI Cricbuzz for cricket data
